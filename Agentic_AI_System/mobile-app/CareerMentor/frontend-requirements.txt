## Installation
To install all required dependencies, run the installation script:
```
chmod +x install-dependencies.sh
./install-dependencies.sh
```

## Starting the Application
After installing dependencies:
```
npm start
```

## Project Structure
- /app/(tabs)/ - Main application tabs and screens
- /components/ - Reusable UI components
- /constants/ - Application constants and theme
- /services/ - API and data services

## Backend Connection
The mobile app connects to the backend API running at:
- Local development: http://localhost:8000
- Device testing: http://192.168.x.x:8000 (replace with your computer's IP)

## Troubleshooting
If you encounter dependency issues, try:
1. Delete node_modules folder and package-lock.json
2. Run `npm install`
3. Run the installation script again
4. Restart the development server
